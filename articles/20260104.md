## 2025年の振り返り

今年は時間の浪費がひどかった。

### 生活

2024年に比べて覚醒継続時間も睡眠時間もなぜか共に増加していて、ばらつきも大きくなっている:

![2024年と2025年の覚醒時間を比較した箱ひげ図とストリッププロット。縦軸は0から40時間、横軸は年。2024年の箱は15時間から20時間の範囲、中央値は約17時間、平均値は約18時間。2025年の箱は15時間から22時間の範囲、中央値は約17.5時間、平均値は約19時間。2025年はひげの範囲が約5時間から33時間までの上下に拡大しており、最大値は約39時間の外れ値を示している。](/images/blog/20260104_awake-duration-comparison-2024-vs-2025.webp)
![2024年と2025年の睡眠時間を比較した箱ひげ図とストリッププロット。縦軸は0から20時間、横軸は年。2024年の中央値と平均値は約10時間弱付近にあるのに対し、2025年はそれらがわずかに上昇して10時間付近にある。また、データのばらつきを示すひげの上限は、2024年が約17.5時間であるのに対し、2025年は約19時間近くまで上に伸びており、長時間睡眠方向へのデータの広がりが増大している。両年とも20時間付近に数点の外れ値がある。](/images/blog/20260104_sleep-duration-comparison-2024-vs-2025.webp)

というかそもそも2025年分の標本数が `292` しかない（インフルエンザで1週間程の抜けがあるとはいえ）。
1日の長さが平均的に `28時間弱 → 29時間` と増大しているのは大きい。
ただ前年通り、生活時間を日本標準時に合わせるための努力は継続していて、覚醒継続時間のばらつきはその調整による影響とも考えられそう（好き勝手に起き続け寝続けをしたらそりゃ継続時間は規則的になる）。

1日や月の平均歩数は前年から据え置きとなっていたので、運動量自体は維持できていそう。
また、1日および月における記録上の最多歩数もこの2025年に更新した。

そしてなにより、外出先での正体不明の失神などがあったため、可及的速やかに健康診断か何かを受けなければならない。

##### OS の乗り換え

まず、メイン PC の OS を [openSUSE Leap 15.6](https://get.opensuse.org/leap/15.6) から [Fedora Linux](https://fedoraproject.org/) 42 ([Workstation Edition](https://fedoraproject.org/workstation)) に乗り換えた。
理由は主に以下の通り:

- openSUSE Leap 15.6 が年末に EOL を迎える（これはいつも通り）
- 15.7 は提供されず、16.0 がリリースされる（まずメジャーアップデートが怖い）
- ALP (Adaptable Linux Platform) を基盤とする方向っぽい（不変 OS はあまり好きじゃない）
  - デスクトップ用途から方針転換？
- X11 から Wayland への完全移行（Wayland で致命的な不具合があったので X11 を使い続けていた）
  - ただ、結局移住した Fedora では Wayland がうまく動作している（そもそも X11 が廃止されている）。
- openSUSE の YaST が廃止（個人的には openSUSE の大きな魅力だった）
- 結果として openSUSE の先行きが不透明に感じてきた
- そもそも3年半使い続けた OS 環境は色々と不安定になってきていた

少し冒険してみようということで Fedora、Debian、openSUSE Slowroll / Tumbleweed あたりを検討した結果、
Fedora に決定した。

構成は Fedora Workstation + Btrfs + Snapper + GNOME + Wayland という感じにした。
多少トラブルがありながらも数日でひととおりの環境構築を終えた。

##### 大掃除と模様替え

次に毎秋恒例の大掃除を実施したのだが、同時に部屋の模様替えも行った。

まず設計に4日かかった。考慮したことは次の通り:

- PC の吸気口や排熱、メンテナンス性
- 掃除保守性（犬の毛の動きなども考慮）
- 空調効率（夏が壊滅的だった）
- 電源の位置
- 音の指向性（寝室が近いため）
- 将来的なマルチモニターの可能性（実際12月に買った）

他にも要件があり最終的に2つの案に絞り込んだ。
どちらも一長一短だったが、最終的には視覚的圧迫感が少ない案を採用した。

掃除に関しても例年以上に大規模に実施した。
4年間使用している PC 内部の清掃も初めて行った。
