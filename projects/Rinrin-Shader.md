自己満足で開発した、マインクラフト統合版向けのシェーダーパック。

## ダウンロード

GitHub の[リリース一覧](https://github.com/Rinrin0413/Rinrin_Shader/releases)からダウンロードできます。

## 対応環境

- プラットフォーム: Minecraft for Windows
- バージョン: `v1.16.100 以前` 又は `32bit版 1.18.12 以前`

## 主な機能

- 新鮮な影
- 真新しい色の光
- 幻想的な空
- リアルな雲
- 擬似的な反射表現
- 透き通った水
- 揺れる草木
- 風に揺れる草ブロック

## 既知の不具合

- 雨天時、空から見た地表が白すぎる。
- 明るすぎる太陽。
- 草木の不自然な揺れ方。
- 発光する草木。
